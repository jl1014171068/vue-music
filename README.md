# vue-music

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

###### 前端：Vue.js2.0
* Vue.js
* Vue-Router
* Vuex
* axios
* Mint UI

###### 数据来源
* easy-mock
* QQ-Music
